# 🕊️ Pigeon Express

A luxury landing page for a fictional premium messenger pigeon service. Built as a single-file responsive website with modern CSS, glassmorphism design, and interactive animations.

## 🎨 Design Highlights

- **Luxury aesthetic** — Deep navy background with gold accents and noise texture
- **Glassmorphism cards** — Frosted glass panels with backdrop blur effects
- **Smooth animations** — Scroll reveal, parallax mouse tracking, floating hero image
- **Fully responsive** — Looks flawless from iPhone SE to 4K displays
- **Accessibility** — Respects `prefers-reduced-motion`, semantic HTML, proper alt text
